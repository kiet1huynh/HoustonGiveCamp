About

GiveCamp is a weekend-long event where technology professionals from designers, developers and database administrators to marketers and web strategists donate their time to provide solutions for non-profit organizations. Since its inception in 2007, the GiveCamp program has provided benefits to hundreds of charities, worth millions of dollars of developer and designer time in services!

FAQ

FAQ #1 - How much does it cost a charity?

Nothing! All work completed during the event is provided at no charge.

FAQ #2 - Do developers have to support the project after the event?

There are no strings attached. What happens at GiveCamp stays at GiveCamp. The code stays with the charity - and developers are not allowed to charge charities for work completed.

FAQ #3 - What happens if we don't finish?

It is the goal of each event to properly scope the projects for completion. If the team is unable to complete the project it is possible that a few individuals may volunteer to wrap up the work at no charge, but this is not guaranteed.
